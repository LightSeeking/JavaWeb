一、内边距——padding

  • 一个值，四个方向边距一样
  • 两个值，就是上下，左右
  • 三个值，就是上，左右，下
  • 四个值，就是上，右，下，左
  • 背景色蔓延到内边距
  • padding-top,padding-left,padding-right,padding-bottom 用于分别设置单一方向的内边距

二、边框——border

  • border:边框粗细 样式 颜色;
  • 样式：solid-实线，double-双实线，dashed-线段线，dotted-点状线
  • 背景色蔓延到边框
  • border-top,border-left,border-right,border-bottom 用于分别设置单一方向的边框
  • border是一个复合属性，可以拆分为： border-width,border-style,border-color PS:上述三个独立属性都可以分别设置四个方向，类似于内边距

三、外边距——margin

1.外边距的基本属性

  • margin和padding一样，也可以独立设置四个方向
  • margin-top,margin-left,margin-right,margin-bottom 用于分别设置单一方向的外边距
  • 背景色不会蔓延！！！
  • 一般会使用*{margin:0;}来清除所有边距，用来重新设置
  • PS:margin可以取负值的！！！

2.屏幕居中的方法

  • margin:0 auto;可以实现横向居中

3.外边距特性

  1. 兄弟关系，两个盒子垂直外边距与水平外边距问题:
     垂直方向，外边距取最大值
     水平方向，外边距会进行合并处理
  2. 父子关系，给子加外边距，但作用于父身上了，怎么解决？
    1. 子的margin-top===>父的padding-top,不过注意父盒子高度变动
    2. 给父盒子设置一个边框即可
    3. 让子盒子或者父盒子加浮动
    4. 父盒子加一个overflow:hidden;即可
