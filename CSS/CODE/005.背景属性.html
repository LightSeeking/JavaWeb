个数 属性 描述 说明
1 background-color 背景颜色 background-color:red;
2 background-image 背景图片 background-image:url();
3 background-repeat 背景图片的平铺 background-repeat:no-repeat/repeat（默认） /repeat-x（铺满x轴）/repeat-y;
4 background-position 背景图片的定位 background-position:水平位置 垂直位置;可以给负值
5 background-attachment 背景图片的固定 background-attachment:scroll(滚动)/fixed(固定，固定在浏览器窗口里面，固定之后 就相当于浏览器窗口了);
可以简写成background
