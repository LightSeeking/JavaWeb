一、浮动元素的高度自适应

  • 父元素不写高度时，子元素写了浮动后，父元素会发生高度塌陷
  • 方法1：给父元素添加声明overflow:hidden;(缺点：会隐藏溢出元素)
  • 方法2：在浮动元素下方添加空格元素，并给该元素添加声明：clear:both;height:0;overflow:hidden;
  • （缺点：在结构里增加了空标签，不利于代码可读性，且降低了浏览器的性能）
  • 方法3：万能清除浮动法
  • 选择符:after{content:" ";clear:both;height:0;visibility:hiddden;/voverflow:hidden;}
  •                     比如：
                    .box::after{
                        content: "";
                        clear: both;
                        /*默认元素是行内的，使用块元素*/
                        display: block;
                        width:0;
                        height: 0;
                        /*
                            display:none;是不占位隐藏
                            visibility:hidden;是占位隐藏
                        */
                        visibility: hidden;
                    }

二、伪元素

  • 1）:after(与content属性一起使用，定义在对象后的内容)
  • 如：div:after{content:url(logo.jpg);}
  • div:after{content:"文本内容";}
  • 2）:before:与content属性一起使用，定义在对象前的内容
  • 如：div:before{content:"在其前放内容";}
  • 3）:first-letter:定义对象内第一个字符的样式
  • 4）:first-line:定义对象内第一行文本的样式
  • PS:一般是两个冒号，一个也行
