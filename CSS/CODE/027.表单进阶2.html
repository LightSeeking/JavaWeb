一、下拉菜单

  • select标签里面的option标签就是选项
  • select支持的属性：size是显示的选项数量，默认一个是能看size个，选择还只能选一个
    mutiple，可以选多个,要按住shift或ctrl键
  • option支持的属性：value是提供后端的需要用到的value值
    selected,要显示的值，默认是第一个选项

二、文本域

  • <textarea cols="30" rows="10">提前设置好的文本，可以接着写，也可以删掉</textarea>
  • cols是可以输入的列数，row是可以输入的行数
  • 一般使用css设置文本域的宽和高
  • placeholder可以设置预置文本
  • 默认值value是写在双标签内部这里的默认值不要随便换行，否则可能会给输入框预置一个换行符
  • 在css中，可设置resize的值，避免文本域被随意放大缩小

三、字段集

  • <fieldset><legend>缺口展示的内容</legend></fieldset>
  • 在fieldset里面，legend外面，写的是框里面的内容
  • 可以使用css控制字段集的样式，对fieldset和legend都可以
