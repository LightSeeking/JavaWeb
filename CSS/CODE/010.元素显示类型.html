元素类型的分类，依据CSS的显示

块元素(block element)

  • A)块级元素在网页中就是以块的形式显示，所谓 块状就是元素显示为矩形区域
  • B)默认情况下，块级元素都会占据一行；默认情况 下，块级元素会按顺序自上而下排列
  • C)块级元素都可以定义自己的宽度和高度
  • D)块级元素一般都作为其他元素的容器，它可以 容纳其他内联元素和其他块级元素，我们把这种 容器比喻成一个盒子

行内（内联）元素

  • A)内联元素的表现形式是始终以行内 住诸葛进行显示；横着排
  • B)内联元素没有自己的形状，不能定义它的宽和高， 它显示的宽度，高度只能根据所包含内容的高度和 宽度来确定，它最小内容单元也会呈现矩形形状
  • C)内联元素也会遵循盒模型基本规则，但是对于margin 和padding个别属性值不生效

行内块元素

  • 内联块状元素(inlin-block)就是同时具备 内联元素、块状元素的特点

例如:div p ul li il li dl dt dd h1-h6等

例如:a b em i span strong等

例如:img input等

  • 1.块元素
  • display:block;
  • display:list-item;
  • p里面一般只放文本，不放其他标签
  • 2.行内元素
  • display:inline;
  • 可以设置左右边距，上下无效
  • 3.行内块元素
  • display:inline-block;
  • 4.元素类型互相转换
  • 不同类型元素之间是有间距的，故可以转换来消除间距
  • 使用：display:block/inline/inline-block/none;
  • display:none;实现隐藏效果
