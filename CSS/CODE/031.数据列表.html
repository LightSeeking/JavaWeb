Datalist:选项列表

例：<input type="url" list="url_list" name="link"/>
<datalist id="url_list">
  <option lable="W3School" value="http://www.W3School.com.cn"/>
  <option lable="Google" value="http://www.google.com"/>
  <option lable="Microsoft" value="http://www.microsoft.com"/>
</datalist>

提示：option元素永远要设置value属性
