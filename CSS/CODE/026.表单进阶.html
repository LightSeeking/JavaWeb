一、单选框

  • <input type="radio">
  • name的值用于分组，同一组只能选一个
  • 使用lable标签，存储文本，为单选框提供文本，使用id关联
  • input里面的checked属性，提供默认勾选,属性和值一样，可以省略不写 
  • 示例：
<input type="radio" name="a" id="first" checked="checked">
<label for="first">单选框</label>

二、复选框（多选）

  • <input type="checkbox">
  • name用于js数据传输
  • 其他类单选框
  • 多选框的默认选择checked可以有多个
  • 示例： <input type="checkbox" name="b" id="cb1" checked> <label for="cb1">复选框</label>

三、上传文件和隐藏字段

  • (1)<input type="file">
  • (2)<input type="image" src="">就是一个图片样式的提交按钮
  • PS:提交等要建立在form表单中
  • (3)<input type="hidden " value="">隐藏按钮
  • 隐藏按钮是为了给后端看的，value就是给后端的数据
  • disabled属性可以禁用组件，readonly是只读属性
