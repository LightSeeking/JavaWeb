自适应

网页布局中经常要定义元素的宽和高。但很多时候我们希望元素能够自己 根据窗口或者子元素自动调整，这就是自适。

宽度自适应高度自适应
  • （1）宽度自适应
  • 元素宽度的默认值为auto
  • 宽度不写或者写成auto就是自适应了
  • 多使用于导航栏，通栏布局
  • （2）高度自适应
  • 元素高度的默认值{height:auto;}
  • 设置自适应，如果内容过少，会导致上下两个模块贴合过近， 不美观，可以设置最小高度min-height；
  • （3）拓展
  • min-height;min-width;max-height;max-width
