1.autofocus属性

给文本框、选择框、或者按钮加上该属性，当打开页面时，该控件自动获得国标焦点，一个页面只能有一个

2.required属性

验证输入不能为空

3.Multiple:

可以输入一个或者多个值，每个值之间用逗号分开

例如：<input type="email" multiple/

还可以应用于file

4.pattern

将属性值设为某个格式的正则表达式，在提交时会检查其内容是否符合给格式

例如：<input pattern = "[0-9][A-Z]{3} title="输入内容：一个数与三个大写字母" placeholder = '输入内容：一个数与三个大写字母'
