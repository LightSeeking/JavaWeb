一、字体模块：@font-face

  • 1.@font-face是CSS3中的一个模块，他主要是把自己定义的Web字体嵌入到你的网页中
  • 2.@font-face的语法规则：
  •     @font-face{
        font-family:<YourWebFontName>;
        src:<source>[<format>][<source>][<format>]*;
        [font-weight:<weight>];
        [font-style:<style>];
    }
  • 3.@font-face语法说明：
  • （1）YourWebFontName:此值值得就是你自定义的字体名称，最好是使用你下载的默认字体， 他将被引用到你的Web元素中的font-family。如“font-family:"YourWebFontName";”
  • （2）source:此值指的是你自定义的字体的存放路径，可以是相对路径也可以是绝对路径
