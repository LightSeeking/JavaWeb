一、UI元素状态伪类选择器概述

  • E:enabled匹配所有用户界面（form表单）中处于可用状态的E元素
  • E:disabled匹配所有用户界面（form表单）中处于不可用状态的E元素
  • E:checked匹配所有用户界面（form表单）中处于选中状态的E元素 一般用于checkbox，不过先appearance: none;清除默认样式
  • E:focus聚焦时会匹配此选择器
  • E::selection匹配E元素中被用户选中或处于高亮状态的部分 一般用于文本的选取效果
