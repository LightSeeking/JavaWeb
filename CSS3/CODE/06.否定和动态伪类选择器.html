一、否定和动态伪类选择器概述

  • 1.否定伪类选择器
  • E:not(s)（IE6-8浏览器不支持:not()选择器）
  • 匹配所有不匹配简单选择符s的元素E

  • 案例分析：

    .case ul li:not(:first-child){
    background-color: lightcyan;
}
  • 2.动态伪类选择器
  • E:link
  • 链接伪类选择器
  • 选择匹配的E元素，而且匹配元素被定义了超链接并未被访问过。常用于链接锚点上
  • E:visited
  • 链接伪类选择器
  • 选择匹配的E元素，而且匹配元素被定义了超链接并已被访问过。常用于链接锚点上
  • E:active
  • 用户行为选择器
  • 选择匹配的E元素，且匹配元素被激活。常用于链接锚点和按钮上
  • E:hover
  • 用户行为选择器
  • 选择匹配的E元素，且用户鼠标停留在元素E之上。IE6及以下浏览器仅支持a:hover
