一、属性选择器概述

  • 1.E[atrr]:只使用属性名，但没有确定任何属性值；
  • 2.E[attr="value"]:指定属性名，并指定了该属性的属性值；
  • 3.E[attr~="value"]:指定属性名，并且具有属性值，此属性是一个词列表，并且 以空格隔开，其中此列表中包含了一个value词，而且等号前面的"~"不能不行。
  • 扩展知识
  • 4.E[attr^="value"]:指定了属性名，并且有属性值，属性值是以value开头的；
  • 5.E[attr$="value"]:指定了属性名，并且有属性值，属性值是以value结束的；
  • 6.E[attr*="value"]:指定了属性名，并且有属性值，属性值中包含了value；
  • 实例分析
  • div[class]{
    background-color: bisque;
}
    表示选择带有class属性的div
