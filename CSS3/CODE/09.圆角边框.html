一、圆角边框-上

  • boder-radius
  • 可以使用px或者百分比
  • 只有一个值，是四个角一样
  • 有两个值，左上右下，左下右上一致
  • 有三个值，左上，左下右上，右下一致
  • 有四个值，从左上角顺时针
  • 只实现一个方向，可以设置其他方向为0，或者使用
    border-top-left-radius
    border-top-right-radius
    border-bottom-left-radius
    border-bottom-right-radius

二、圆角边框-下

  • border-radius:30px/60px;表示水平走30px，竖直走60px
  • 设置四个角:
    border-radius:10px 20px 30px 40px/10px 20px 30px 40px
  • border-radius: 50%;可将正方形设置成一个圆
