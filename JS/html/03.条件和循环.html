一、条件分支语句

  • 1.if的书写和使用同java
  • 2.switch 
    switch(需要匹配的值){
    case 值1:
        ...
        break
    case 值2:
        ....
        break
    ...
    default:
        ...
}
  • 不写break会从第一个满足条件的选项开始向下穿透，直到遇到break

二、循环结构语句

  • 1.while循环: 
    while(条件){
    会被重复执行的代码
}
  • 2.dowhile循环: do{ 会被重复执行的代码 } while(条件)
  • 与while的区别：无论是否满足添加，循环体都会被执行一次，而while不会
  • 3.for循环： 
    for(定义初始变量 ; 条件判断 ; 改变初始值){
    重复执行的代码
}
