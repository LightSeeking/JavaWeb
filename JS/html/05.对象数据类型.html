一、对象数据类型概述

  • 对象就是键值对的集合
  • 1.定义：var obj = {键:值,键:值,...}
  • 2.操作
  • (1)增： 
    方式一:obj.name='lightseeking'
方式二:obj['age'] = 18
  • (2)删： 
    方式一:delete obj.name
方式二:delete obj['age']
  • (3)改： 
    方式一:obj.name='LightSeeking'
方式二:obj['age']=24
  • (4)查： 
    方式一:console.log(obj.name)
方式二:console.log(obj['age'])
